Listas Não Ordenadas

São escritas com a tag ul. É utilizada para representar uma lista de elementos que não tem necessidade de ordenação. Ex.:Lista de compras

  • Banana
  • Maçã
  • Abacate
  • uvas

Listas Ordenadas

São escritas com a tag ol. São utilizadas para elementos que são ordenados. Ex.: Receitas

  1. Acordar
  2. Tomar café da manhã
  3. Ir para o curso Geração Tech

Tipos de listas ordenadas

O atributo 'type' controla a renderização do marcador do tipo lista
